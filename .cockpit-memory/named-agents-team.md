Mythic roster finalized (Baz's constraints: no Turkish names, no Hermes): Atlas=planner/opus, Apollo=frontend/opus, Vulcan=backend/opus, Argos=security-reviewer/sonnet, Huginn=scout/sonnet, Calliope=copy/sonnet. Each agent's markdown definition (in ~/.claude/agents, project .claude/agents overrides) carries role+persona+model+tool-perms; role & persona come FROM the definition, so assigning an agent to a card supersedes the manual Role/Persona selectors (those are the None/fallback path). Wired into Swarm via shared/named-agents.ts kernel, V6 migration (kanban_cards.agent slug column), NamedAgentsService, card-editor Agent selector; worker spawns with the agent's voice and model (claude --model opus for Atlas/Apollo/Vulcan). Single definition, three consumers: cards, terminal subagents, and worker sub-spawns. Shipped in v0.1.29.
