The 'cockpit' wordmark (hero + left rail) uses Space Grotesk, self-hosted at public/fonts/space-grotesk-latin.woff2 (single variable woff2, latin subset, ~22KB, OFL). Added via @font-face + a display-font token in tokens.css. This is CSP-compatible: index.html CSP allows font-src 'self' data: only — no Google Fonts / gstatic at runtime, so any new webfont must be bundled locally, not linked. Vite copies public/ into out/renderer/ (served at /fonts/... , returns 200 on localhost).
