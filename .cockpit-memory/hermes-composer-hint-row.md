Keyboard shortcut hints moved from inline placeholder to a persistent <kbd> row below the composer capsule. Placeholder stays clean: "Message Hermes…" vs the old verbose version. Capsule itself (attach + textarea + send) uses a single warm focus ring around the whole row instead of three separate boxes, for a calmer read. This pattern is now the established composer convention.
